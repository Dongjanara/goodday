  <h2> 기획 의도 </h2>
    <li> 여(기), 좋은 데이 사이트는 사용자들이 자신의 동네 또는 근거리의 여행 장소를 손쉽게 찾아볼 수 있도록 하는 것이 목표입니다. </li>
    <li> 여행을 찜하고 구매하는 기능을 통하여 여행 구매에 대한 편의를 제공하는 것이 첫번째 목표입니다. </li>
    <li> 추천인 아이디 기능을 통하여 포인트를 제공하고 구매 시 사용 할 수 있도록 하고, 주변의 다른 사람들도 가입을 유도하는 홍보 기능을 만들었습니다. </li>
    <li> 리뷰기능을 도입하여 사용자들에게 더욱 실질적인 정보를 제공합니다. </li>
    <li> 여행 경험을 공유하고 다른 사용자들의 의견을 들을 수 있는 커뮤니티 활동을 지원하여, 신뢰성 있는 리뷰와 다양한 의견을 통해 사용자들이 더 나은 여행 결정을 할 수 있도록 합니다. </li>
    <li> 사용자들에게 더욱 풍부하고 진솔한 여행 경험을 제공하는 것 또한 목표로 하고 있습니다. </li>
    <li> 최종적으로는 이런 진실성 있는 리뷰와 지역 사회와 연결된 다양한 여행 장소 정보를 통해 더 많은 사용자를 유입 시켜 여행 커뮤니티를 형성하고자 합니다. </li>
</form>

    <h2> [팀 공통 목표] </h2>
      <li> MVC2 모델을 완전히 이해하는 것을 목표로 합니다. </li>
      <li> 웹 전체 페이지를 이용자들이 쉽게 접근할 수 있도록 직관적으로 표현합니다. </li>
      <li> 상품 카테고리 및 상세 페이지는 Oracle SQL DB를 사용하여 일관성 있게 관리합니다. </li>
      <li> 회원가입 시 개인정보 보호를 위해 암호화를 적용합니다. </li>
      <li> 상품 리뷰 및 댓글 기능을 통해 커뮤니티를 활성화합니다. </li>
      <li> 매일 프로젝트 진행 후 화면 설계서 및 기능 명세서를 기반으로 프로젝트 진행 상황을 공유합니다. </li>
    <h2> [개인 목표] </h2> 
  → 상품조회
  <li> 목표 : 사용자가 다양한 조건으로 상품을 검색할 수 있는 기능을 제공합니다. </li>
  <li> 구현 계획 : 카테고리별 검색 기능을 개발하여, 사용자가 원하는 상품을 빠르고 정확하게 찾을 수 있도록 합니다. </li>
  <br>
  → 장바구니
  <li> 목표 : 사용자가 자신이 담은 상품을 쉽게 확인 할 수 있도록 기능을 제공합니다. </li>
  <li> 구현 계획 : 구매 총 수량, 합계 금액을 한눈에 쉽게 가독성을 높여서 구현합니다. 장바구니 비우기를 누를 시, 전체 장바구니가 비워지도록 하며, 개별 상품 삭제 또한 가능하도록 합니다. </li>
  <br>
  → 카카오지도 API
  <li> 목표 : 상품별 위치를 카카오지도 API를 통하여 사용자에게 쉽게 정보를 제공합니다. </li>
  <li> 구현 계획 : 카카오 지도 API를 학습하여, 상품별 위치를 일관성 있게 나타내도록 합니다. 길 찾기를 누를 시, 목적지가 자동으로 등록이 되도록 합니다. </li>
  <br>
  → 장바구니
  <li> 목표 : 사용자가 자신이 담은 상품을 쉽게 확인 할 수 있도록 기능을 제공합니다. </li>
  <li> 구현 계획 : 구매 총 수량, 합계 금액을 한눈에 보기 쉽게 가독성을 높여서 구현합니다. 장바구니 비우기를 누를 시, 전체가 비워지도록 하며, 개별 상품 삭제도 가능하도록 합니다. </li>
  <br>
  → 찜하기
  <li> 목표 : 사용자가 관심있는 상품을 찜하기 기능을 통하여, 장바구니와는 별개로 저장하도록 합니다. </li>
  <li> 구현 계획 : 장바구니 틀과 동일한 양식으로 구현하고, 목록 전체 비우기, 개별삭제 기능을 통하여 사용자가 쉽게 찜 목록을 관리할 수 있습니다. </li>
  <br>
  → 리뷰기능
  <li> 목표 : 상품 구매를 했을 경우에만, 리뷰 작성을 하도록 합니다. </li>
  <li> 구현 계획 : 사용자는 마이페이지 상 리뷰를 통하여, 내가 쓴 리뷰를 열람하고 수정 및 삭제를 할 수 있습니다. 상품 상세보기 페이지에서도 해당 상품 리뷰를 볼 수 있습니다. </li>
  <br>
  → 댓글기능
  <li> 목표 : 작성된 리뷰에 대한 댓글을 작성할 수 있습니다. </li>
  <li> 구현 계획 : 아이디와 상품번호를 기준으로 리뷰에 대한 댓글을 작성할 수 있습니다. 댓글은 수정 및 삭제도 가능합니다. </li>
